Para desinstalar o Java Runtime Environment: No Painel de Controle, selecione Desinstalar um programa. Selecione o Java 8 Update 171 na lista e clique em Remover. Para alterar o valor de -jdkhome: Navegue até a pasta etc do diretório de instalação do Netbeans IDE. O local default é C:\Program Files\NetBeans 8.2\etc.
Abra o arquivo netbeans.conf em um editor de texto e edite a opção netbeans_jdkhome. Observação: o JavaTM SE Development Kit (JDK) 5.0 e JDK 6 não são suportados em nenhuma plataforma desta release. Você deve ter o JDK 7 Update 10 (ou mais recente) ou JDK 8 no seu sistema para instalar e executar o NetBeans IDE 8.2. Lorem Ypson
